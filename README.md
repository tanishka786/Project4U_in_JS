# 🎯 JavaScript Mini Projects Collection  

This repository contains a set of **beginner-friendly JavaScript projects**, all created using **HTML, CSS, and Vanilla JS**.  
Each folder represents a standalone mini project and the root files act as a **main landing page**.

---

## 🚀 Projects Included

### 1️⃣ Color Changer  
A simple project that lets you **change background colors** using JavaScript.  
Great for practicing event handling and DOM manipulation.  
**Status:** ✔️ Completed  

---

### 2️⃣ BMI Calculator  
Calculate your **Body Mass Index** by entering your height & weight.  
Includes input validation and health category mapping.  
**Status:** ✔️ Completed  

---

### 3️⃣ Digital Clock  
A live digital clock that updates every second.  
Uses `Date()` and `setInterval()` for real-time updates.  
**Status:** ✔️ All requirements fulfilled  

---

### 4️⃣ Guess The Number  
A fun guessing game with hints like **Too High**, **Too Low**, or **Correct!**  
Perfect for learning conditions and loops.  
**Status:** ✔️ All requirements fulfilled  

---

## ▶️ How to Run

### 🔹 Run From Landing Page  
1. Open the main `index.html`  
2. Click the links/buttons to open each mini project  

### 🔹 Run Any Project Directly  
Navigate into its folder (`1-colorChanger`, `2-BMICalculator`, etc.)  
and open the HTML file inside.
